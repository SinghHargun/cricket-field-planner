# cricket-field-planner
Interactive cricket field planner built in Java Swing — place, drag, and manage players on a visual cricket field

## Features
* Click to place batsmen (blue) and bowlers (red) on the field
* Drag players to reposition them
* Click an existing player to remove them
* Enforces max 11 players (6 batsmen, 6 bowlers)
* Prevents player overlap automatically

## Technologies Used
* Java
* Java Swing (GUI)
* Object-Oriented Programming

## How to Run
1. Clone the repository
2. Compile: `javac CricketFieldGUI.java`
3. Run: `java CricketFieldGUI`
